## Opis projektu
W projekcie posłużyłem się bibliotekami m.in. requests, xlsxwriter. Projekt ma za zadanie pobrać dane z API, a następnie 
utworzyć plik w excelu i zapisać tam dane w postaci tabelarycznej oraz wykonać wykres ilości wykonanych poprawnie zadań od id użytkownika. Dodatkowo dodałem wyświetlanie
w konsoli numerów id tych użytkowników, którzy uzyskali najwięcej poprawnych odpowiedzi, jak i tych którzy uzyskali najmniej.
W projekcie posłużyłem się fejkowymi danymi z darmowego REST API "JSONPlaceholder"

## A project description
In my project I imported libraries i.a. requests, xlsxwriter. The purpouse of this program is retrieving data from the API, then create an excel file, save the data
there in tabular form and make a graph of the number of correct tasks from user id. Additionally, In the console is displayed the id numbers of those users who obtained the most correct
answers and those who obtained the least. In my project I used fake data from the free REST API "JSONPlaceholder".
